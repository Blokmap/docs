# Developer Documentation

Welcome to the BlokMap developer documentation! This comprehensive guide will help you understand, contribute to, and extend the BlokMap platform.

## Overview

BlokMap is a modern web application built with:
- **Frontend**: React with TypeScript
- **Backend**: Node.js with Express
- **Database**: PostgreSQL
- **Deployment**: Docker containers on cloud infrastructure

## Quick Start

1. **Clone the repository**
2. **Set up your development environment**
3. **Install dependencies**
4. **Run the development servers**

See the [Quick Setup](/developers/setup) guide for detailed instructions.

## Architecture

BlokMap follows a modern microservices architecture with clear separation between frontend and backend components. Learn more in our [Architecture Overview](/developers/architecture).

## Development Areas

### Frontend Development
- React application with TypeScript

### Backend Development

### Deployment
