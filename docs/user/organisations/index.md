# Organisatiebeheerder

Als **organisatiebeheerder** beheer je een overkoepelende organisatie (bv. een onderwijsinstelling) binnen Blokmap. Vanuit de organisatie kan je:

- locaties koppelen en beheren
- leden beheren
- beheerders en rollen beheren
- groepen aanmaken en toegang toekennen

## Snelkoppelingen

- [Leden beheren](./members.md)
- [Toegangsbeheer](./access/)
- [Groepen beheren](./authorities.md)
- [Toegang beheren](./institution-access.md)

::: info
Organisaties kunnen gebruikt worden om beheer te centraliseren. In veel gevallen krijgen organisatiebeheerders automatisch toegang tot gekoppelde locaties.
:::
