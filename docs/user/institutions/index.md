# Organisatiebeheerder

Als **organisatiebeheerder** beheer je een overkoepelende organisatie (bv. een onderwijsinstelling) binnen Blokmap. Vanuit de organisatie kan je:

- locaties koppelen en beheren
- leden beheren
- beheerders en rollen beheren
- groepen aanmaken en toegang toekennen

:::
