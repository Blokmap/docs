# Reservaties beheren

Als je ervaart dat de capaciteit beperkend is (of je verplicht om andere redenen specifieke inschrijving), kun je reservaties inschakelen op locatieniveau. Wanneer reservaties zijn ingeschakeld, zie je effectief de gemaakte reservaties via je dashboard op een specifieke reservatiepagina.

[SCREENSHOT_PLACEHOLDER: Screenshot of the reservation management list per day]

**Wat kun je op deze pagina?**
- **Lijst inzien:** Bekijk de details van welke studenten geregistreerd staan per openingsdag en tijdslot.
- **Handmatige aanwezigheid:** Voor kleinere locaties is het nuttig om fysieke aanwezigheid handmatig te kunnen bevestigen wanneer de student arriveert, hiervoor is een functionaliteit aanwezig op het overzicht.
- **Annuleren & Verwijderen:** Is de locatie toch gesloten om een afwijkende reden? Of dient een specifieke inschrijving geweigerd te worden? Individuele reservaties annuleren of verwijderen beheer je eveneens via dit scherm.
