# Strafpunten

::: warning Waarschuwing
Het strafpuntensysteem is momenteel in ontwikkeling en is binnenkort beschikbaar.
:::

Wanneer een student niet komt opdagen voor een reservering (een _no-show_), worden er strafpunten aan hun profiel toegekend. Deze strafpunten zijn platformbreed: punten die op de ene locatie zijn verzameld, tellen ook mee op alle andere locaties binnen Blokmap.

Als locatiebeheerder bepaal je echter zelf hoe strikt je deze strafpunten hanteert voor jouw specifieke locatie:

- **In- of uitschakelen**: Kies of je voor jouw locatie wel of geen strafpuntensysteem wilt gebruiken.

- **Geldigheid**: Bepaal hoelang strafpunten meetellen. Je kunt bijvoorbeeld instellen dat punten ouder dan één maand worden genegeerd. Zo voorkom je dat studenten permanent worden uitgesloten van reserveringen.

- **Drempelwaarde**: Stel in vanaf welk aantal actieve strafpunten een student automatisch wordt geblokkeerd voor het maken van nieuwe reservaties op jouw locatie.
