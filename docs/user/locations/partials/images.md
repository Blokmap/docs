Je kunt maximaal vijf afbeeldingen toevoegen. Kies hier een omslagfoto: dit is het eerste beeld dat studenten zien in het overzicht en op de Blokmap-kaart. Je bepaalt zelf de volgorde waarin de overige foto's in de galerij verschijnen.

Een foto toevoegen kan op twee manieren:

- **Uploaden vanaf je toestel (aanbevolen)**: Selecteer een foto uit je galerij of bestanden. De afbeelding wordt veilig op de Blokmap-server opgeslagen.
- **Via een link invoegen**: Voer de URL in van een afbeelding die je wilt tonen. Let op: we hebben geen invloed op de beschikbaarheid van externe links. Als de link niet meer werkt, verdwijnt de afbeelding van de locatiepagina.
